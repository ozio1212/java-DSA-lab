# DSA Laboratories

A repository containing solutions for laboratory tasks from the **Data Structures and Algorithms** course at Wrocław University of Science and Technology (Politechnika Wrocławska).

## General Information
- **Language:** Java
- **Structure:** Each package (`dsaa.labXX`) corresponds to subsequent laboratory sessions.

## Laboratory List

| Package | Topic | Brief Description |
| :--- | :--- | :--- |
| `dsaa.lab01` | Introduction | Java basics, I/O, drawing geometric shapes (pyramid, Christmas tree), and loading documents. |
| `dsaa.lab02` | Singly Linked Lists | Implementation of a custom singly linked list (`OneWayLinkedList`) based on the `IList` interface. |
| `dsaa.lab03` | Doubly Linked Lists | Implementation of a doubly linked list with head and tail references (`TwoWayUnorderedListWithHeadAndTail`). |
